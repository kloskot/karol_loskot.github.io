Imperial College Projects

  • Benign/malignant tumor diagnosis through machine learning in Python (2023)
    A group project with the objective of building a machine learning model capable of diagnosing malignant or benign breast cancer tumors using and trained by medical images. This project used many industry standard libraries such as Keres, TensorFlow, Pandas and SciKit learn and built using Python.
  • Explorative data analysis and visulisation of Starbucks food and drink menu (2023)
    An individual project which explored the nutritional breakdown of starbucks menu through interactive and informative data visulisations developed in R.
  • Memetic Algorithm for solving the knapsack problem (2022)
    Independant project where I developed a multi-memetic AI algorithm tasked at solving the knapsack problem, built using Java and an API developed by the University of Nottingham which implemented approaches such as Davis Bit Hill climbing and stohastic gradient decent.
  • VR Ice Skating mixed reality experience (2023)
    A realistic mixed/virtual reality experience designed using a ROVR treadmill for omnidirectional movment, sensor fans for imitation of wind resistance and live sounds and a working prototype developed in Unity VR for a Oculus Quest 2 headset
  • Online multiplayer puzzle game built using test-driven development (2023)
    A group project with the focus of familiarising ourselves with industry standards of software development. We developed a 3D online escape-room type game using Unity whilst following ISO 9000 standards, test-driven development, industry project developemnt methedologies and automation testing.
  • VR educational game about the UK legal system (2022)
    A year long group project with the objective of educating the user about how the UK legal system works. The user would be punished accordingly to the crime they choose to or choose not to commit. This project was developed in Unity VR and for the Occulus quest 2 headset.
  • Improvment of code and UI and addition of functionality to game Breakout(2022)
    Following industry coding conventions, I fixed and improved the readabiliaty and extendability of the code through the use of code design patterns; better organised the class hierarchy; improved user interface using Java Swing and Java Fx; implemented main menu, scoreboard and extra mechanics to the game. This was done in Java.
  • Recreation of the game Othello (2021)
    Developed in Java, the UI was developed using Java Swing and Java FX and the logic was also written in Java. This project also focused on the use of UML to explain design decisons and XML.
  • Comparison of differnt algorithms and hyperparameters of a robot vaccume cleaner through simulation (2023)
    A simulation of a robot vaccum cleaner tasked with collecting dirt in a room. The intelligent agent and enviroment was visulised using the tkinter library. The focus was to compare two features, first being the hyperparameters of the lidar the robot used to avoid obsticals, the other being the pathfinding algorithm used such as A* and Dijkstra. This project was developed in Java.
  • Autonomous Robot using ROS melodic (2023)
    Fully autonomous robot vacume cleaner capable of navigation and localisation using SLAM, object detection using computer vision, and collision avoidance using lidar developed for a ROS waffle bot. The project was tested both physically and virtually using an enviroment designed in Gazebo.
  • Penetration testing of server with weak security and fixing its flaws (2023)
    Using software such as metasploit and nmap, this project focused on familiarising yourself with such software and identifying the security flaws of a database. Furthermore it included tasks such as creating strong and robust password policies and imlementing changes to the database to make it more secure.
  • Imperial Maker Challenge (2019)
    A 12 week course & project taught at imperial college which taught me skills such as: Use woodwork heavy tools such as Band saw and Drill press, use of laser cutter, programming and use of CAD, use of 3D printer, differnt approaches to product design and idea generation, soldering and how to use a breadboard to build circuits, use of Arduino. For my product I desgined and created a works-like and looks-like prototype of a health monitoring dog collar inspired by FitBit which would record and display roughly the amount of callories the dog has burnt and the amount of steps it took through the use of an accelerometer and gyroscope.
  • Imperial Gold Crest Award (2019)
    A group reaserch project on a randomly assigned field which later was presented at Imperial collage. Me and my team where assigned the field of biomechanics from which we wrote the paper titled: "The biomechanics of Computer gaming". To conduct our research we had frequent meeting with our mentor who was a PHD student of biomechanics and used observation through labeling and recording the movment of differnt hand joins affected by the ergonomics of the mouse as well as the style of grip.
